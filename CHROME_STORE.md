## Name
Video Translator

## Short Description (132 chars)
Real-time video translation — captures tab audio directly, no microphone needed. Supports 12+ languages with bilingual AR/EN interface.

## Detailed Description

**Video Translator** translates any video or audio playing in your browser tab in real-time. No microphone needed — it captures the audio directly from the tab!

### ✨ Key Features

🎧 **Direct Audio Capture**
Captures audio playing in any browser tab — no microphone required. Works with YouTube, TikTok, and any website with video/audio content.

🌍 **12+ Languages**
Chinese, English, Japanese, Korean, French, Spanish, Russian, Turkish, German, Portuguese, Hindi, Arabic.

🗣️ **Bilingual Interface**
Full Arabic and English interface with automatic RTL/LTR support. Switch languages with one click.

🖱️ **Draggable Overlay**
The translation overlay floats over the video and can be dragged anywhere on screen.

📝 **Translation History**
Keeps a log of all recent translations for reference.

🆓 **100% Free**
Uses free APIs — no paid subscriptions or API keys needed.

### 🚀 How to Use

1. Install the extension
2. Start the included backend server (one-time setup)
3. Open any video page
4. Click the extension icon → Select languages → Start Capture
5. Watch real-time translations appear on screen!

### 🛡️ Privacy
- No microphone access required
- Audio processing happens locally
- Minimal permissions
- No data sent to third parties (except translation APIs)

### 📋 Permissions
- `activeTab` — Access current tab for audio capture
- `scripting` — Inject overlay into video pages
- `storage` — Save your preferences
